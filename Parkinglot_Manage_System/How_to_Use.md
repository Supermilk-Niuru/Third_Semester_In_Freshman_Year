# 停车场管理系统 - 使用文档

---

## 一、项目简介

这是一个用 C++ 编写的停车场管理系统，利用面向对象的设计思想，将**车辆（Car）**和**停车场（ParkingLot）**分别封装为两个类，实现了车辆入场、出场缴费、车位管理和历史记录等功能。

---

## 二、文件说明

| 文件名 | 作用 |
|--------|------|
| `Car.h` | 车辆类的头文件，定义车辆的属性和方法 |
| `ParkingLot.h` | 停车场类的头文件，定义停车场的管理逻辑 |
| `main.cpp` | 主程序文件，包含菜单界面和所有交互逻辑 |
| `Makefile` | 编译脚本，简化编译操作 |

> **说明**：由于程序比较小，所有类的实现都直接写在了头文件（.h）中，初学者更容易理解。没有拆分出 .cpp 实现文件。

## 三、函数说明

### 3.1 Car 类（Car.h）—— 车辆

Car 类代表一辆车，封装了车辆的所有信息和对信息的操作。

| 函数 | 返回值 | 功能 | 怎么调用 |
|------|--------|------|----------|
| `Car()` | 无 | **默认构造函数**：创建一个空车对象，入场时间为 0，未缴费 | `Car c;` |
| `Car(string no, string color, string s)` | 无 | **带参构造函数**：传入车牌号、车牌颜色、停车位置，自动记录当前时间为入场时间 | `Car c("京A12345", "蓝色", "东边");` |
| `getPlateNo()` | string | 获取车牌号 | `string s = c.getPlateNo();` |
| `getPlateColor()` | string | 获取车牌颜色 | `string s = c.getPlateColor();` |
| `getEntryTime()` | long | 获取入场时间（Unix 时间戳，秒数） | `long t = c.getEntryTime();` |
| `getSide()` | string | 获取停车位置（"东边" 或 "西边"） | `string s = c.getSide();` |
| `isPaid()` | bool | 判断是否已缴费（true=是，false=否） | `if (c.isPaid()) { ... }` |
| `setPaid(bool p)` | void | 设置缴费状态（true=已缴费） | `c.setPaid(true);` |
| `getTimeStr()` | string | 获取格式化的入场时间字符串（如 "Wed Jul 8 14:30:25 2026"） | `cout << c.getTimeStr();` |
| `getMinutes()` | double | 计算停车时长（分钟），已缴费则返回 0 | `double m = c.getMinutes();` |
| `getFee()` | double | 计算停车费用（元），3元/小时，不足1小时按分钟计费 | `double f = c.getFee();` |

---

### 3.2 ParkingLot 类（ParkingLot.h）—— 停车场

ParkingLot 类代表停车场，管理车位和所有车辆的入场出场。

| 函数 | 返回值 | 功能 | 怎么调用 |
|------|--------|------|----------|
| `ParkingLot(int east, int west)` | 无 | **构造函数**：初始化停车场，东边 east 个车位，西边 west 个车位 | `ParkingLot lot(50, 50);` |
| `getEastFree()` | int | 获取东边剩余可用车位数（总车位 - 已停车数） | `int n = lot.getEastFree();` |
| `getWestFree()` | int | 获取西边剩余可用车位数 | `int n = lot.getWestFree();` |
| `getEastTotal()` | int | 获取东边总车位数 | `int n = lot.getEastTotal();` |
| `getWestTotal()` | int | 获取西边总车位数 | `int n = lot.getWestTotal();` |
| `getCarCount()` | int | 获取当前停在场内的车辆总数 | `int n = lot.getCarCount();` |
| `enterCar(Car car)` | bool | **车辆入场**：将一辆车加入停车场。成功返回 true，车位满了返回 false | `bool ok = lot.enterCar(c);` |
| `exitCar(string plateNo)` | Car | **车辆出场**：根据车牌号找到车辆并移出。返回被移出的车对象，找不到返回空车 | `Car c = lot.exitCar("京A12345");` |
| `findCar(string plateNo)` | int | **查找车辆**：根据车牌号查找。返回数组下标（0,1,2...），找不到返回 -1 | `int i = lot.findCar("京A12345");` |
| `getCar(int index)` | Car | 通过下标获取车辆对象（配合 findCar 使用） | `Car c = lot.getCar(0);` |
| `showParkedCars()` | void | **打印当前在场车辆列表**到屏幕 | `lot.showParkedCars();` |
| `showHistory()` | void | **打印历史停车记录**到屏幕 | `lot.showHistory();` |
| `getHistoryCount()` | int | 获取历史记录总条数 | `int n = lot.getHistoryCount();` |
| `getHistory(int index)` | Car | 通过下标获取某条历史记录 | `Car c = lot.getHistory(0);` |

### 3.3 main.cpp 中的函数

这些是主程序中的界面和控制函数。

| 函数 | 功能 | 在哪里被调用 |
|------|------|--------------|
| `clearScreen()` | 清屏（根据系统自动选择 cls 或 clear 命令） | 每个界面的开头 |

## 四、如何编译运行

### 4.1 环境要求

- 操作系统：Mac / Linux / Windows
- 编译器：g++（支持 C++11 及以上）
- 终端：任意支持命令行的终端（macOS 终端、Windows PowerShell 等）

### 4.2 编译方法

#### 方法一：使用 Makefile（推荐）

打开终端，进入项目目录：

```bash
cd parking-system

# 编译（生成可执行文件 parking）
make

# 运行
make run

# 或者直接运行
./parking

# 清理（删除编译生成的文件）
make clean
```

#### 方法二：直接用 g++ 命令

```bash
cd parking-system

# 编译
g++ -Wall -std=c++11 -o parking main.cpp

# 运行
./parking
```

#### Windows 用户特别注意

Windows 上编译后生成的是 `parking.exe`，运行命令是：

```bash
g++ -Wall -std=c++11 -o parking.exe main.cpp
parking.exe
```

## 六、收费标准说明

| 规则 | 计算方式 |
|------|----------|
| 基础单价 | **3 元 / 小时** |
| 不足 1 小时 | 按分钟计费，**0.05 元 / 分钟**（3 ÷ 60 = 0.05） |
| 公式 | `费用 = 停车分钟数 × 0.05` |

**举例**：
- 停车 30 分钟 → 30 × 0.05 = **1.50 元**
- 停车 1小时（60分钟）→ 60 × 0.05 = **3.00 元**
- 停车 2小时15分钟（135分钟）→ 135 × 0.05 = **6.75 元**

## 七、面向对象设计说明

```
┌──────────────────────┐     ┌──────────────────────────┐
│        Car           │     │       ParkingLot         │
│──────────────────────│     │──────────────────────────│
│ - plateNo : string   │     │ - eastTotal : int        │
│ - plateColor : string│     │ - westTotal : int        │
│ - entryTime : long   │     │ - eastCount : int        │
│ - side : string      │     │ - westCount : int        │
│ - paid : bool        │     │ - parkedCars[100] : Car  │
│──────────────────────│     │ - carCount : int         │
│ + Car()              │     │ - history[500] : Car     │
│ + getPlateNo()       │     │ - historyCount : int     │
│ + getPlateColor()    │     │──────────────────────────│
│ + getSide()          │     │ + enterCar(Car) : bool   │
│ + getMinutes()       │◄────│ + exitCar(string) : Car  │
│ + getFee()           │包含 │ + findCar(string) : int  │
│ + getTimeStr()       │     │ + showParkedCars()       │
│ + setPaid(bool)      │     │ + showHistory()          │
└──────────────────────┘     └──────────────────────────┘
```

- **封装**：Car 的成员变量都是 private，只能通过 public 的 getter/setter 访问
- **包含关系**：ParkingLot 内部用 Car 数组来管理车辆（一个停车场包含多辆车）
- **职责分离**：Car 只负责车辆自身信息，ParkingLot 负责管理逻辑
