# 停车场管理系统 - 使用文档



## 一、项目简介

这是一个用 **C++** 编写的停车场管理系统，运行在控制台中。
利用面向对象的设计思想，将**车辆（Car）**和**停车场（ParkingLot）**分别封装为两个类。

### v3 新特性

| 特性 | 说明 |
|------|------|
| 🚗 **长租车 / 临时车** | 入场时选择车辆类型，长租车出场不收费 |
| ⭐ **白牌车免费** | 白色军用车辆出场不收费 |
| 📱 **收款码** | 收费时弹出收款码图片，扫码付款 |

### 功能介绍

| 功能 | 说明 |
|------|------|
| 🚗 **车辆入场** | 录入车牌号、选择车牌颜色、选择车辆类型、选择停车位置 |
| 🚙 **车辆出场（缴费）** | 计算停车时长和费用，确认后放行，白牌车/长租车免费 |
| 📋 **在场车辆查询** | 显示当前停车场内所有车辆（含类型标记） |
| 📜 **历史记录查询** | 显示所有已出场车辆记录 |

### 收费规则

| 规则 | 说明 |
|------|------|
| ⛔ **黄牌车禁入** | 大型车辆车身过大，禁止入场 |
| ⭐ **白牌车免费** | 白色军用车辆出场不收费（v3新增） |
| 🏠 **长租车免费** | 月保长租车辆出场不收费（v3新增） |
| 🔋 **绿牌车九折** | 新能源电动车停车费打九折 |
| ⚡ **充电桩** | 东西两边各 25 个，绿牌车入场可选 |
| 🆓 **15分钟免费** | 停车不满 15 分钟免费出场 |

## 二、文件说明

| 文件名 | 作用 |
|--------|------|
| `Car.h` | 车辆类的头文件，定义车辆的属性和方法 |
| `ParkingLot.h` | 停车场类的头文件，定义停车场的管理逻辑 |

## 三、函数说明

### 3.1 Car 类（Car.h）—— 车辆

Car 类代表一辆车，封装了车辆的所有信息和对信息的操作。

#### 成员变量

| 变量 | 类型 | 说明 |
|------|------|------|
| `plateNo` | string | 车牌号，如 "京A88888" |
| `plateColor` | string | 车牌颜色：蓝色/黄色/白色军用车辆/港牌/澳门牌/台湾牌/绿色 |
| `entryTime` | long | 入场时间（Unix 时间戳，秒数） |
| `side` | string | 停车位置："东边" 或 "西边" |
| `paid` | bool | 是否已缴费 |
| `needCharging` | bool | 是否需要充电桩（绿牌车专用） |

#### 函数列表

| 函数 | 返回值 | 功能 | 怎么调用 |
|------|--------|------|----------|
| `Car()` | 无 | **默认构造函数**：创建空车对象，`entryTime=0`，`paid=false`，`needCharging=false` | `Car c;` |
| `Car(no, color, s, needCharge)` | 无 | **带参构造函数**：传入车牌号、颜色、位置、是否需要充电桩。第4个参数可省略（默认false）。入场时间自动设为当前时间 | `Car c("京A12345", "绿色", "东边", true);` |
| `getPlateNo()` | string | 获取车牌号 | `string s = c.getPlateNo();` |
| `getPlateColor()` | string | 获取车牌颜色 | `string s = c.getPlateColor();` |
| `getEntryTime()` | long | 获取入场时间戳 | `long t = c.getEntryTime();` |
| `getSide()` | string | 获取停车位置（"东边"或"西边"） | `string s = c.getSide();` |
| `isPaid()` | bool | 判断是否已缴费 | `if (c.isPaid()) { ... }` |
| `getNeedCharging()` | bool | 判断是否需要充电桩 | `if (c.getNeedCharging()) { ... }` |
| `setPaid(bool p)` | void | 设置缴费状态 | `c.setPaid(true);` |
| `setNeedCharging(bool n)` | void | 设置是否需要充电桩 | `c.setNeedCharging(true);` |
| `getTimeStr()` | string | 获取入场时间字符串，如 "Wed Jul 8 14:30:25 2026" | `cout << c.getTimeStr();` |
| `getMinutes()` | double | 计算停车时长（分钟），已缴费返回 0 | `double m = c.getMinutes();` |
| `getFee()` | double | **计算停车费**：不满15分钟免费 → 绿牌九折 → 正常计费 | `double f = c.getFee();` |
| `getDiscountInfo()` | string | 获取优惠说明文字（出场界面显示用） | `cout << c.getDiscountInfo();` |

#### getFee() 计算逻辑

```
停车分钟数 < 15 → 返回 0.00（免费）
       ↓
正常计算：分钟数 × 0.05 元
       ↓
如果是绿色（新能源）→ 乘以 0.9（九折）
       ↓
返回最终费用
```

### 3.2 ParkingLot 类（ParkingLot.h）—— 停车场

ParkingLot 类代表停车场，管理车位、充电桩和所有车辆的入场出场。

#### 成员变量

| 变量 | 类型 | 说明 |
|------|------|------|
| `eastTotal` | int | 东边总车位数 |
| `westTotal` | int | 西边总车位数 |
| `eastCount` | int | 东边已停车辆数 |
| `westCount` | int | 西边已停车辆数 |
| `eastChargingTotal` | int | 东边充电桩总数（固定 25） |
| `westChargingTotal` | int | 西边充电桩总数（固定 25） |
| `eastChargingUsed` | int | 东边已占用的充电桩数 |
| `westChargingUsed` | int | 西边已占用的充电桩数 |
| `parkedCars[100]` | Car[] | 当前停着的车辆数组 |
| `carCount` | int | 当前车辆数 |
| `history[500]` | Car[] | 历史出场记录数组 |
| `historyCount` | int | 历史记录条数 |

#### 函数列表

| 函数 | 返回值 | 功能 | 怎么调用 |
|------|--------|------|----------|
| `ParkingLot(east, west)` | 无 | **构造函数**：初始化停车场和充电桩（默认各 25 个） | `ParkingLot lot(50, 50);` |
| `getEastFree()` | int | 东边剩余车位数（负数时返回 0） | `int n = lot.getEastFree();` |
| `getWestFree()` | int | 西边剩余车位数（负数时返回 0） | `int n = lot.getWestFree();` |
| `getEastTotal()` | int | 东边总车位数 | `int n = lot.getEastTotal();` |
| `getWestTotal()` | int | 西边总车位数 | `int n = lot.getWestTotal();` |
| `getCarCount()` | int | 当前在场车辆总数 | `int n = lot.getCarCount();` |
| `getEastChargingFree()` | int | 东边空闲充电桩数 | `int n = lot.getEastChargingFree();` |
| `getWestChargingFree()` | int | 西边空闲充电桩数 | `int n = lot.getWestChargingFree();` |
| `getEastChargingTotal()` | int | 东边充电桩总数 | `int n = lot.getEastChargingTotal();` |
| `getWestChargingTotal()` | int | 西边充电桩总数 | `int n = lot.getWestChargingTotal();` |
| `enterCar(Car)` | int | **车辆入场**，返回结果码（0成功/1黄牌禁入/2车位满/3充电桩满） | `int r = lot.enterCar(c);` |
| `exitCar(string plateNo)` | Car | **车辆出场**，移除车辆并释放充电桩，返回被移出的车 | `Car c = lot.exitCar("京A12345");` |
| `findCar(string plateNo)` | int | 查找车辆，返回数组下标，-1 表示没找到 | `int i = lot.findCar("京A12345");` |
| `getCar(int index)` | Car | 通过下标获取车辆对象 | `Car c = lot.getCar(0);` |
| `getHistoryCount()` | int | 获取历史记录总条数 | `int n = lot.getHistoryCount();` |
| `getHistory(int index)` | Car | 通过下标获取某条历史记录 | `Car c = lot.getHistory(0);` |
| `showParkedCars()` | void | 打印当前在场车辆列表（含 ⚡ 充电标记） | `lot.showParkedCars();` |
| `showHistory()` | void | 打印历史停车记录 | `lot.showHistory();` |

#### enterCar() 返回值说明

| 返回值 | 含义 | 说明 |
|--------|------|------|
| `0` | ✅ 入场成功 | 车辆正常停入 |
| `1` | ⛔ 黄牌禁入 | 黄牌车车身过大，禁止入场 |
| `2` | ❌ 车位已满 | 对应侧没有空车位了 |
| `3` | ❌ 充电桩已满 | 绿牌车需要充电但对应侧充电桩已满 |

#### enterCar() 逻辑流程图

```
用户选择车牌颜色
       │
       ▼
┌─── 是黄色？───→ 返回 1（禁止入场）
│
│   否
│   ▼
┌─── 是白色？───→ 跳过"车位已满"检查
│
│   否
│   ▼
┌─── 对应侧还有车位？─── 否 ──→ 返回 2（车位已满）
│
│   是
│   ▼
┌─── 是绿色且需要充电桩？
│   │   是
│   │   ▼
│   ┌─ 对应侧充电桩有空闲？── 否 ──→ 返回 3（充电桩满）
│   │   是
│   ▼   ▼
  执行入场：车位计数+1，充电桩计数+1（如果需要）
  返回 0（成功）
```

## 四、如何编译运行

### 4.1 环境要求

- 操作系统：Mac / Linux / Windows
- 编译器：g++（支持 C++11 及以上）
- 终端：任意支持命令行的终端

### 4.2 编译方法

#### 方法一：使用 Makefile（推荐）

```bash
cd parking-system

# 编译
make

# 运行
make run

# 或者直接运行
./parking

# 清理
make clean
```

#### 方法二：直接用 g++ 命令

```bash
cd parking-system

# 编译
g++ -Wall -std=c++11 -o parking main.cpp

# 运行
./parking
```

#### Windows 用户

```bash
g++ -Wall -std=c++11 -o parking.exe main.cpp
parking.exe
```

### 5.3 车辆出场 + 缴费（选 2）

按 `2` 回车，进入出场缴费。v3 新增了**白牌车/长租车免费**和**收款码**功能。

#### 出场流程总览

出场流程分为三种情况：

| 情况 | 处理方式 |
|------|----------|
| 🏠 **长租车** | 直接免费出场，不收费 |
| ⭐ **白牌车** | 直接免费出场，不收费 |
| 🆓 **不满15分钟** | 自动免费出场 |
| 💰 **其他临时车** | 显示收款码 → 扫码付款 → 确认出场 |

#### 长租车 / 白牌车出场（免费）

```
请输入出场车辆的车牌号：京A77777

  ========== 车辆信息 ==========
  车牌号：京A77777
  车牌颜色：蓝色

## 六、收费标准说明

| 规则 | 说明 |
|------|------|
| 基础单价 | **3 元 / 小时** |
| 不足 1 小时 | 按分钟计费，**0.05 元 / 分钟**（3 ÷ 60） |
| **不满 15 分钟** | **🆓 免费** |
| **绿牌新能源车** | **🔋 打九折**（原价 × 0.9） |
| **白牌军用车辆** | **⭐ 免费**（v3 新增） |
| **长租月保车辆** | **🏠 免费**（v3 新增） |

### 计费逻辑（按优先级）

```
是否是白牌车？       → 是 → 免费 0 元
         ↓ 否
是否是长租车？       → 是 → 免费 0 元
         ↓ 否
停车分钟数 < 15      → 免费 0 元
         ↓ 否
正常费用 = 分钟数 × 0.05 元
         ↓
是绿牌车？           → 是 → 费用 × 0.9（九折）
         ↓ 否
返回最终费用
```

## 七、收款码设置说明（v3 新增）

收费时系统会自动弹出收款码图片，供扫码付款。

### 设置步骤

1. 将你自己的微信/支付宝收款码截图保存为 **`qrcode.png`**
2. 把 `qrcode.png` 放到项目目录 `parking-system/` 下
3. 收费时系统会自动弹出该图片

### 支持的打开方式

| 操作系统 | 命令 |
|----------|------|
| **Ubuntu / Linux** | `xdg-open qrcode.png` |
| **macOS** | `open qrcode.png` |
| **Windows** | `start qrcode.png` |

> 💡 如果没有弹出图片，可以手动打开项目目录下的 `qrcode.png`
> 💡 如果没有收款码，系统仍然正常运行，只是不会弹出图片

## 八、面向对象设计说明

```text
┌──────────────────────────┐     ┌──────────────────────────────────┐
│          Car              │     │           ParkingLot             │
│──────────────────────────│     │──────────────────────────────────│
│ - plateNo : string       │     │ - eastTotal : int                │
│ - plateColor : string    │     │ - westTotal : int                │
│ - entryTime : long       │     │ - eastCount : int                │
│ - side : string          │     │ - westCount : int                │
│ - paid : bool            │     │ - eastChargingTotal : int (25)   │
│ - needCharging : bool    │     │ - westChargingTotal : int (25)   │
│──────────────────────────│     │ - eastChargingUsed : int         │
│ + Car()                  │     │ - westChargingUsed : int         │
│ + Car(no,color,side,...) │     │ - parkedCars[100] : Car          │
│ + getPlateNo()           │     │ - carCount : int                 │
│ + getPlateColor()        │     │ - history[500] : Car             │
│ + getMinutes()           │◄────│ - historyCount : int             │
│ + getFee()               │ 包含│──────────────────────────────────│
│ + getTimeStr()           │     │ + getEastFree() : int            │
│ + getWestFree() : int    │     │ + getWestFree() : int            │
│ + getDiscountInfo()      │     │ + getEastChargingFree() : int    │
│ + setNeedCharging(bool)  │     │ + getWestChargingFree() : int    │
└──────────────────────────┘     │ + enterCar(Car) : int (0/1/2/3)  │
                                 │ + exitCar(string) : Car          │
                                 │ + findCar(string) : int          │
                                 │ + showParkedCars()               │
                                 │ + showHistory()                  │
                                 └──────────────────────────────────┘

```
### 设计原则

- **封装**：Car 和 ParkingLot 的成员变量都是 private，只能通过 public 的 getter/setter 访问
- **包含关系**：ParkingLot 内部用 `Car` 数组来管理车辆（一个停车场包含多辆车）
- **职责分离**：Car 只负责车辆自身信息（属性 + 计算费用），ParkingLot 负责管理逻辑（车位、充电桩、入场出场）
